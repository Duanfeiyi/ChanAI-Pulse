# Changelog

All notable changes to ChanAI Pulse will be documented in this file.

## v1.0.0 - Initial Release Candidate

### Added

- MATLAB-based desktop research prototype for AI-based predictive channel modeling.
- Channel characterization module for loading and visualizing channel characteristics.
- Channel generation module with native stochastic channel generation workflow.
- Channel prediction and training module.
- TCN, LSTM, and GRU model selection support.
- Synthetic demo data for public GitHub quick-start validation.
- Local measured dataset policy and measured-data audit documentation.
- Low-risk extracted helper functions under `core/`.
- Smoke test scripts for environment and repository validation.
- GitHub collaboration documentation for student teams.
- GUI manual test checklist.
- Release checklist and dataset policy.
- MATLAB Compiler diagnosis and packaging solution notes.

### Notes

- Full measured datasets are not included in the public repository.
- MATLAB installer packaging is not included in this RC because MATLAB Compiler is not visible in the current local MATLAB installation.
- This release is intended as a MATLAB desktop research prototype.
