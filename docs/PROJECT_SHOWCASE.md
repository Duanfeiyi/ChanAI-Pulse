# ChanAI Pulse Project Summary

ChanAI Pulse is a MATLAB research prototype that links local channel-data visualization, lightweight synthetic generation and baseline DPSD time-series prediction in one three-page desktop App.

## Current demonstration value

- Shows a traceable MATLAB workflow from compatible MAT input through legacy characteristic displays, optional synthetic augmentation, baseline training and held-out evaluation.
- Provides TCN, LSTM and GRU baselines with chronological validation/test separation.
- Keeps public demos synthetic and supplies data-contract and converter utilities for future dataset work.

## Current boundaries

The project does not yet provide verified all-band/all-scenario coverage, complex-H prediction, QuaDRiGa, Base Models, online adaptation, MIMO prediction or a web platform. Generation and evaluation limitations are documented in [Open Issues](OPEN_ISSUES_AND_REFACTOR_ROADMAP.md).

## Research direction

The long-term direction is dynamic wideband complex-H prediction with carefully controlled data contracts, public/authorized datasets, Base Model adaptation and eventually MIMO space-time-frequency tensors. This is a roadmap, not a claim about current code.
