# 平台交互逻辑设计基线

**最后更新：** 2026-07-17  
**状态：** 在线适配主流程已确认作为后续交互设计基线；具体 UI 尚未实现。完整路线见 `07_confirmed_master_plan.md`。

## 1. 当前平台任务流

```text
导入数据
→ 数据处理
→ 数据可视化
→ 6GPCM-lite 数据扩充
→ 针对当前数据从零训练 TCN / LSTM / GRU
→ 进行功率域预测
→ 展示 RMSE、DS CDF 等结果
```

当前流程的主要特征：

- 每个上传数据集通常重新训练模型；
- 没有预训练 Base Model 与场景 Adapted Model 的区分；
- 输入和输出主要是功率域实数向量；
- 数据扩充在用户理解中接近固定步骤；
- 旧流程继续保留为可复现 Power Baseline。

## 2. 未来离线阶段

```text
QuaDRiGa 仿真数据
+ DeepMIMO 等仿真数据
+ 获得授权的开放实测复数 CSI
→ 统一成复数 H 数据契约
→ 按场景、轨迹、位置或测量会话划分 Train / Validation / Test
→ 训练并验证通用 Base Model
→ 保存模型版本、归一化参数、配置、指标和适用范围
→ 发布到平台的 Base Model Registry
```

“通用”必须有明确适用边界。第一版建议限定为动态宽带 SISO，而不是宣称一个模型已经覆盖任意频段、场景和天线规模。

## 3. 未来在线用户任务流（已确认方向）

```text
导入动态宽带复数信道数据
→ 数据类型识别与质量检查
→ 统一转换为复数 H 数据格式
→ 信道特性分析与可视化
→ 读取场景、频率、带宽、采样间隔和天线元数据
→ 推荐或选择兼容的 Base Model
→ 先执行 Base Model 直接预测，建立零样本基线
→ 判断是否需要数据扩充与模型适配
→ （可选）使用 QuaDRiGa / 6GPCM-lite 生成训练扩充数据
→ 冻结 Base Model 主体并调整少量参数
→ 使用 Validation 检查适配是否有效
→ 若适配变差则回退 Base Model
→ 保存当前场景的 Adapted Model
→ 预测未来复数 H
→ 计算点级、分布级和通信系统级指标
→ 展示、导出并记录完整实验
```

如果上传数据只有 PDP、DPSD 或其他功率表示，应进入旧 Power Baseline，并明确提示不能完成完整复数 H 预测。

## 4. 在线适配的含义

在线适配不是用户手工修改神经网络权重，而是平台利用用户已经观测到的历史 H 序列，自动构造监督窗口并对少量可训练参数做小学习率更新。

示例：

```text
H1 ... H600     → 适配训练
H601 ... H800   → 适配验证与 early stopping
H801 ... H1000  → 独立测试（正式实验时不得参与适配）
H1001 以后      → 真正未来预测目标
```

适配的默认安全策略：

1. Base Model 只读，不被覆盖；
2. 冻结通用特征编码器和大部分预测主干；
3. 只训练输出层、最后预测块或 Adapter；
4. 使用小学习率、较少 epoch、early stopping 和最佳 Validation checkpoint；
5. 比较适配前后 Validation 指标；
6. 适配无收益或变差时自动回退；
7. Adapted Model 单独保存，并记录来源、配置和适用场景。

## 5. 建议向用户提供的三种模式

### 直接预测

- 不更新网络参数；
- 更新必要的数据格式、尺度和归一化；
- 作为零样本泛化基线。

### 快速适配（建议默认）

- 冻结大部分 Base Model；
- 只训练输出层、最后模块或 Adapter；
- 适合有限用户数据。

### 深度微调（高级）

- 解冻更多模型层；
- 需要更多数据、训练时间和更严格的过拟合检查；
- 不作为第一版默认路径。

## 6. 数据扩充在新流程中的位置

数据扩充改为可选决策：

- 数据充分：不扩充，直接适配；
- 数据有限：只根据适配训练段校准 QuaDRiGa，并将生成数据加入适配训练；
- 数据极少：优先使用最相近 Base Model，不强行微调；
- 6GPCM-lite：用于快速、备用或基线扩充，不承担主要研究结论；
- Validation、Test 和真正未来数据不得用于生成器校准。

## 7. 未来三页交互映射

### 第一页：数据与信道特性

```text
导入 H / CIR / CTF / PDP
→ 数据契约验证
→ 相位连续性、时间顺序和元数据检查
→ 展示幅度、相位、PDP、DS、Doppler、天线相关性
```

### 第二页：信道生成与训练数据准备

```text
选择 None / 6GPCM-lite / QuaDRiGa
→ 配置或根据训练段估计场景参数
→ 生成连续复数信道轨迹
→ 比较仿真与真实训练段
→ 决定是否加入适配训练集
```

### 第三页：模型选择、适配与预测

```text
选择或自动推荐 Base Model
→ 运行直接预测基线
→ 选择适配等级
→ 训练并验证 Adapter / 微调层
→ 比较适配前后性能
→ 预测未来复数 H
→ 保存 Adapted Model 和实验记录
```

## 8. 交互安全与提示

- 适配开始前显示哪些数据段会参与训练、验证和测试；
- 显示 Base Model ID、适用范围和输入维度；
- 明确显示“Base Model 未修改”以及新 Adapted Model 的保存位置；
- 数据不足、相位不连续或维度不兼容时拒绝深度微调，并给出原因；
- 展示适配前后指标，不仅显示训练损失；
- 所有长任务支持取消，失败时恢复按钮、模型和页面状态。
