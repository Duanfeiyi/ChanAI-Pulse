# 数据、模型与科研技术思路

**最后更新：** 2026-07-17  
**状态：** 复数 H、QuaDRiGa 主数据源、Base Model + Adapter 主线已确认；具体数据参数和模型结构待实验确定。

## 1. 预测对象

最终研究目标不是“预测实部和虚部”与“预测 H”二选一，而是：

> 预测未来复数信道 H；Re/Im 双通道是第一版使用普通实数网络表示复数 H 的工程方案。

对于宽带 MIMO：

```text
H[t, frequency, rx_antenna, tx_antenna] : complex
```

网络表示可采用：

```text
X[t, frequency, rx_antenna, tx_antenna, 2]
最后一维 2 = Real / Imaginary
```

第一版建议以动态宽带 SISO 频域 `H(t,f)` 为默认任务，同时保留 CIR，并使用 FFT/IFFT 保持频域与时延域之间的可追踪转换。

## 2. 数据来源定位

### QuaDRiGa

- 主要研究级仿真来源；
- 必须生成连续轨迹、复数系数、正确时间采样和场景元数据；
- 用于多场景预训练和训练数据扩充；
- 仿真效果必须在独立真实 Test 上验证。

### DeepMIMO 等射线追踪数据

- 补充具有几何环境和 MIMO 结构的数据；
- 属于仿真或射线追踪，不得描述为实测；
- 需建立独立 adapter 和数据来源标签。

### 开放实测 CSI

使用前检查：复数系数、时间戳、连续轨迹、采样间隔、载频、带宽、天线、子载波、相位一致性、许可证和隐私边界。

### 6GPCM-lite

- 保留为 Lightweight Baseline；
- 用于单元测试、GUI、快速原型、消融和 QuaDRiGa 不可用时的备用；
- 生成器的复数 CIR 有继续利用价值；
- 当前简单功率混合增强不作为新复数 H 主线的核心科学方法。

## 3. 数据契约待扩展项

现有 ChanAIs 契约需要为复数预测补充或明确：

- 轴顺序与每一轴语义；
- `domain = frequency | delay`；
- 原始 H、估计 H-hat 与生成 H 的区别；
- 快拍时间戳、采样间隔和轨迹；
- 载频、带宽、子载波间隔；
- Tx/Rx 天线几何、极化和阵列配置；
- phase reference、相位校准和同步质量；
- 速度、运动方向和动态散射体信息；
- 数据来源、许可、隐私和处理历史；
- 是否具备时间预测、空间任务或仅特性分析资格。

## 4. 静态与动态数据路由

| 类型 | 可以做什么 | 不能默认做什么 |
| --- | --- | --- |
| `dynamic_temporal` | 未来 H 预测、Doppler、时变分析 | 无 |
| `spatial_track` | 空间插值、位置外推、信道地图 | 不应伪装成时间预测 |
| `static_drop` | 特性分析、生成建模、统计比较 | 非平凡未来时间预测 |
| `static_repeated_measurement` | 噪声、估计稳定性、重复性分析 | 将测量噪声等同信道演化 |

完全静止场景仍有 CIR、PDP、DS 和频率选择性；理想情况下其慢时间变化和 Doppler 为零。

## 5. 数据划分与增强

必须先按完整轨迹、区域、场景或测量会话划分，再构造滑窗：

```text
真实 Train → 可用于训练、生成器校准、模型适配
真实 Validation → 模型选择、early stopping
真实 Test → 最终独立报告
Synthetic → 只能进入训练或预训练
```

禁止：

- 使用完整真实数据校准生成器后再随机切分；
- 将相邻快拍随机分散到 Train 和 Test；
- 将独立静态 drop 排列成虚假时间序列；
- 用未来目标数据参与 Adapter 训练；
- 用合成 Test 替代真实场景结论。

## 6. 建议的模型顺序

### 第一层：必须保留的简单基线

- Last-value / Persistence；
- 线性外推或移动平均；
- AR；
- Kalman。

### 第二层：当前神经网络基线升级

- TCN；
- LSTM；
- GRU；
- 输入输出改为 Re/Im 双通道，但保留旧功率模型版本。

### 第三层：通用与适配

- Base Model 直接预测；
- 归一化与场景条件适配；
- 输出头或最后预测块微调；
- Adapter；
- 元学习或少样本适配；
- 数据充分时的部分/完整微调。

### 第四层：复杂结构

- 固定规模 MIMO；
- 角度—时延或其他压缩域预测；
- 注意力、Transformer、复杂值网络或物理约束；
- 只有在可靠基线与统一 Benchmark 后再进入。

## 7. 评价体系

### 点级

- Complex NMSE；
- 复数相关系数；
- 幅度误差；
- 相位误差；
- 单步与多步误差。

### 分布与物理级

- 经验 PDP；
- 经验 DS CDF；
- K-S / Wasserstein；
- Doppler spectrum；
- 天线相关性与空间统计。

### 通信系统级

- 可达速率或容量；
- 波束赋形损失；
- 预编码性能；
- 相比过时 CSI 的收益；
- 推理延迟与适配成本。

CDF 只能作为宏观统计验证，不能替代逐样本 complex NMSE 和通信系统性能。

## 8. 推荐 A/B 实验

在同一独立真实 Test 上比较：

```text
从零真实训练
QuaDRiGa-only 训练
真实 + QuaDRiGa 训练
QuaDRiGa 预训练 + 真实微调
Base Model 零样本预测
Base Model + 用户数据快速适配
```

## 9. 关键科学风险

- 仿真到真实 domain gap；
- 实测相位参考不一致导致 Re/Im 不可预测；
- 频率 bin、时延 bin 和物理轴混用；
- 固定 200/500 bin 掩盖不同带宽和分辨率；
- 用户数据过少导致微调过拟合；
- 适配破坏 Base Model 的通用能力；
- 只报告曲线相似或 CDF，不报告点级与系统级收益；
- “通用”“全场景”等表述超过实际实验范围。
