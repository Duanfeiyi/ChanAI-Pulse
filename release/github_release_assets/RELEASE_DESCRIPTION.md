# ChanAI Pulse v1.0.0

ChanAI Pulse is an AI-based Predictive Channel Modeling Platform for 6G multi-band and multi-scenario communications.

This v1.0.0 release is a MATLAB desktop research prototype.

## Important

This release requires MATLAB and required toolboxes. It is not a standalone executable.

No private measured datasets are included in this release.

## Included

- MATLAB App source code
- Channel characterization workflow
- Channel generation workflow
- Channel prediction and training workflow
- TCN / LSTM / GRU model selection support
- Synthetic demo data
- Documentation and collaboration guides
- MATLAB App Package release notes

## MATLAB Requirements

Recommended:

- MATLAB R2022b or later

Required toolboxes:

- Deep Learning Toolbox
- Signal Processing Toolbox
- Statistics and Machine Learning Toolbox

Recommended:

- Communications Toolbox
- 5G Toolbox

## Dataset Policy

Only synthetic demo data is included. Private measured datasets are not part of this public release and must not be uploaded to GitHub.

## Future Plan

A standalone installer may be provided in a future release after MATLAB Compiler packaging is resolved.
