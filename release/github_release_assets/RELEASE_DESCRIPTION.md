# ChanAI Pulse v1.0.0

Historical v1.0 release-description draft. Current implementation scope is documented in the repository README; this file does not establish verified all-frequency, all-scenario, Complex-H, or standalone-installer capability.

This v1.0.0 release provides a MATLAB desktop research prototype and MATLAB App Package.

## Important

This release requires MATLAB and required toolboxes. It is not a standalone executable.

No private measured datasets are included in this release.

## Source Code

The public source code is available at:

```text
https://github.com/Duanfeiyi/ChanAI-Pulse
```

## Included

- MATLAB App source code
- MATLAB App Package asset
- Channel characterization workflow
- Channel generation workflow
- Channel prediction and training workflow
- TCN / LSTM / GRU model selection support
- Synthetic demo data
- Documentation and collaboration guides
- Dataset and benchmark planning documents

## MATLAB Requirements

Recommended:

- MATLAB R2022b or later

Required toolboxes:

- Deep Learning Toolbox
- Signal Processing Toolbox
- Statistics and Machine Learning Toolbox

Recommended:

- Communications Toolbox
- 5G Toolbox

## Dataset Policy

Only synthetic demo data is included. Private measured datasets, local raw archives, extracted previews, and large experiment outputs are not part of this public release and must not be uploaded to GitHub.

## Future Plan

Future releases will continue to improve the MATLAB App Package and documentation. Long-term planning includes the ChanAIs Dataset, reproducible benchmark tasks, baseline models, and possible standalone packaging after MATLAB packaging requirements are resolved.
