# ChanAI Pulse v1.0.0 安装说明

## 1. 这是什么版本

这是 ChanAI Pulse v1.0.0 的 MATLAB App Package 版本。

文件名：

```text
ChanAI_Pulse_v1.0.0.mlappinstall
```

注意：这不是独立 exe。用户仍然需要安装 MATLAB 和必要工具箱。

## 2. MATLAB 版本要求

推荐：

- MATLAB R2022b 或以上

## 3. 必需工具箱

- Deep Learning Toolbox
- Signal Processing Toolbox
- Statistics and Machine Learning Toolbox

## 4. 推荐工具箱

- Communications Toolbox
- 5G Toolbox

## 5. 如何安装 `.mlappinstall`

方法一：

1. 打开 MATLAB；
2. 双击 `ChanAI_Pulse_v1.0.0.mlappinstall`；
3. MATLAB 会弹出安装窗口；
4. 点击安装；
5. 安装完成后，在 MATLAB 的 Apps 里找到 ChanAI Pulse。

方法二：

在 MATLAB 命令行运行：

```matlab
matlab.apputil.install("ChanAI_Pulse_v1.0.0.mlappinstall")
```

## 6. 如何启动

安装完成后：

1. 打开 MATLAB；
2. 进入 Apps；
3. 找到 ChanAI Pulse；
4. 点击启动。

## 7. 如何使用 synthetic demo data

本版本包含 synthetic demo data，用于快速测试加载和可视化：

```text
demo_sub6_scenario1.mat
demo_mmwave_scenario2.mat
```

这些数据不是实测数据，不能作为论文 benchmark 证据。

## 8. 为什么不包含真实测量数据

真实测量数据属于内部研究数据，需要单独授权、脱敏和版本管理。因此 GitHub Release 不包含真实测量数据。

## 9. 常见问题

### Q: 这个 App 能脱离 MATLAB 运行吗？

不能。`.mlappinstall` 需要 MATLAB 环境。

### Q: 为什么不是 exe？

独立 exe 需要 MATLAB Compiler。当前 v1.0.0 发布的是 MATLAB App Package，不是 standalone installer。

### Q: 打开后提示缺少工具箱怎么办？

请安装必需工具箱，或联系项目维护者确认环境。

### Q: 能否使用自己的 `.mat` 数据？

可以尝试，但数据结构需要与 App 支持的格式兼容。建议先使用 synthetic demo data 测试流程。
